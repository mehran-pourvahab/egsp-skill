
# EGSP v1 — Engineering Grade Software Production Skill

This package contains the EGSP engineering governance standard for AI-assisted development.

## Contents

SKILL.md
  Canonical engineering standard.

docs/
  Expanded documentation of engineering practices.

templates/
  Reusable engineering artifacts (ADR template, engineering report schema).

checklists/
  Operational checklists used during development and deployment.

## How to Use

1. Load SKILL.md as a system prompt or skill for AI agents (Claude, Cursor, Copilot, Antigravity).
2. Use templates/ for ADRs and engineering reports.
3. Use checklists/ before implementation and deployment.

The canonical engineering policy is defined in **SKILL.md**.
