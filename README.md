# EGSP Skill — Engineering-Grade Software Production

> A structured, AI-ready skill for engineering-grade software production — covering architecture, security, observability, and lifecycle management.

Available in **two formats** depending on your use case:

---

## 📄 Single-File Version (Compact Runtime)
**File:** [`egsp_skill.md`](./egsp_skill.md)

Best for: **Chatbots**, quick integrations, and environments where a single file is preferred.

- Self-contained, no dependencies
- Paste directly into your AI system prompt or skill loader
- Minimal overhead

---

## 📦 Modular Version (Operational Project Package)
**Folder:** [`EGSP/`](./EGSP/)

Best for: **AI Agents**, multi-agent pipelines, and production deployments.

- Structured with docs, templates, and checklists
- Follows the Antigravity Skill Package standard
- Designed for agent skill loaders and orchestration frameworks

---

## Quick Start

### For Chatbots
Copy the contents of [`egsp_skill.md`](./egsp_skill.md) into your system prompt.

### For AI Agents
Point your skill loader to the [`EGSP/SKILL.md`](./EGSP/SKILL.md) entry point.
